함수
	- 함수(funtion)는 특정 작업을 수행하는 
	명령어들의 모음에 이름을 붙인 것.
	- 함수는 작업에 필요한 데이터를 전달받을 수 있으며, 
	작업이 완료된 후에는 작업의 결과를 호출자에게 반환할 수 있다.
: 수학에서의 함수는 반드시 값을 리턴한다. 
프로그래밍에서는 두가지의 함수가 종류가 있다 . : 	값을 리턴하는 함수 (= 리터럴 함수_)
					값을 리턴하지 않는 함수 (= 프로시져 함수)

함수의 예 ) 
	print()
	input()
	abs(), ...
	
	- 함수 안의 명령어들은 실행하려면 함수를 호출(call)하면 된다
		함수 사용 하는 것을  함수 사용이라 함.

함수정의 

함수 라이브러리 생성

함수 사용의 장점 
	- 프로그램 안에서 중복된 코드를 제거한다. 
	- 복잡한 프로그래밍 작업을 더 간단한 작업들로 분해할 수 있다.
	- 함수는 한번 만들어지면 다른 프로그램에서도 재사용될 수 있다.
	- 함수를  사용하면 가독성이 증대되고, 유지 관리도 쉬워진다.

함수의 이름 
	- 함수의 목적을 설명하는 동사 또는 동사+명사를 사용
	
함수 호출 
	- 함수를 사용하려면 함수를 호출(call)하여야 한다. 

함수는 재사용이 가능하다 . 여러번 호출 가능. 

함수 작성의 예 
	

함수를 이용할 때 주의할 점 
	- 파이썬 인터프리터는 함수가 정의되면 함수 안의 문장들은 즉시 실행하지 않는다. 
	- 함수 정의가 아닌 문장들은 즉시 실행하게 된다.
